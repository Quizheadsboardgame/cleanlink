# **App Name**: PortalFlow

## Core Features:

- Stock Order Form Management: Enable users to efficiently input and manage stock order details, including their name, order date, chosen site, and a list of specific items with quantities.
- Dynamic Item Addition: Provide a flexible interface for users to dynamically add and remove multiple item entries to their stock order, ensuring scalability for various order sizes.
- WhatsApp Order Submission: Facilitate quick and direct order submission by generating a pre-filled WhatsApp message with all order details, ready for dispatch to a predefined recipient.

## Style Guidelines:

- The chosen theme is dark, providing a sleek and modern backdrop that enhances focus on content.
- Primary color: A sophisticated indigo blue (#6E76F5). This color will be used for interactive elements, highlights, and calls to action, providing a professional and energetic contrast against the dark background.
- Background color: A very dark, desaturated purplish-grey (#16151C). This deep tone forms the core of the sleek dark mode, ensuring readability and a premium feel.
- Accent color: A vibrant cyan blue (#14ADFF). This analogous color will be used sparingly for key notifications, subtle glows, and elements requiring significant visual pop, ensuring dynamism without overpowering the primary.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, tech-forward, and slightly bold presence in titles and section headers.
- Body font: 'Inter' (sans-serif), providing a clean, highly readable, and neutral foundation for all body text, ensuring clarity and objective communication.
- Utilize a consistent set of minimal, outline-style vector icons that reinforce the modern and streamlined aesthetic. Prioritize icons that clearly communicate functionality and aid in quick visual scanning.
- Implement a clean, grid-based layout with generous use of 'darkspace' to provide visual breathing room. Content sections should be organized into modular cards or panels for easy navigation and dynamic arrangement.
- Incorporate subtle, performant micro-animations for interactions such as hover effects on cards and buttons, smooth transitions for opening/closing modals, and dynamic content loading to enhance the feeling of responsiveness and fluidity.
